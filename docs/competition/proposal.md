# MoonLogLens 一页项目申报书

## 项目名称

MoonLogLens：MoonBit 轻量级结构化日志解析与查询引擎

## 项目简介

MoonLogLens 是一个面向 MoonBit 生态的轻量级结构化日志解析、查询与聚合基础库。项目支持 logfmt 风格日志解析、带位置的错误诊断、字段过滤、全文关键字匹配、字段存在性查询和按字段计数聚合，可作为日志采集与查询系统、自动化构建日志分析、服务运行状态排查等场景的基础组件。

## 项目方向与适用场景

日志是基础软件系统中最常见的可观测数据来源之一。MoonLogLens 先提供一个小而完整的 MoonBit 日志处理内核，使开发者能够在 MoonBit 程序中快速解析结构化日志、筛选故障事件、统计服务分布，并为后续日志采集、查询系统和可观测性工具奠定基础。

适用场景包括：

- 服务运行日志的本地快速筛选；
- 自动化构建、测试、CI 输出的结构化分析；
- 边缘设备或轻量服务中的日志摘要统计；
- 后续扩展为日志采集、索引和可视化查询系统。

## 计划实现的核心功能

- 解析 `level=ERROR service=api msg="request timeout"` 等 logfmt 风格日志。
- 支持多行日志解析，并在错误中保留行列位置。
- 支持查询表达式 `level:ERROR service:api text:"timeout" has:trace_id`。
- 提供字段读取、消息读取、过滤匹配和按字段计数聚合 API。
- 提供命令行演示、单元测试、README、设计文档、验收清单和 CI 配置。

## 原创性说明

本项目为原创项目，不是移植已有开源项目。项目围绕 MoonBit 生态中的结构化日志解析与查询需求重新设计数据结构、扫描器、查询语法、错误诊断、测试和 CLI 示例。项目采用 Apache-2.0 开源许可证。

## 技术路线

解析器采用确定性单遍扫描，不依赖外部库。扫描器逐字符维护当前位置、当前键、当前值、是否处于引用值，以及引用值中的转义状态。查询器复用相同的轻量扫描思路，将查询文本转换为字段等值、全文包含和字段存在三类条件。聚合器使用数组保存首次出现顺序，保证输出稳定。

项目结构：

- 根包：核心库 API；
- `cmd/main`：可运行 CLI 示例；
- `*_test.mbt`：黑盒测试；
- `docs/competition`：申报与提交材料；
- `.github/workflows`：CI 验证配置。

# MoonCSV Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a complete MoonBit CSV parsing, serialization, validation, CLI demo, and competition documentation project.

**Architecture:** The root package exposes a pure library with parser, serializer, table helpers, and diagnostics. `cmd/main` depends on the root package and demonstrates the public API with an embedded sample. Documentation and CI make the repository ready for public contest review.

**Tech Stack:** MoonBit 0.1.20260529, MoonBit core library, `moon test`, `moon fmt`, `moon info`, GitHub Actions.
